# Knight vs Sceletors

Welcome to **Knight vs Sceletors**, a top-down action adventure built with Python and Pygame!  
Play as a brave knight, explore a tile-based dungeon map, defeat enemies, collect gold, and upgrade your gear in the shop menu.

This project was created as part of a university assignment.

## Features

- **Top-down combat**: Fight enemies using melee weapons and ranged attacks (bow + arrows).
- **Enemy AI**: Enemies chase, attack, and some of them shoot.
- **Loot and economy**: Earn gold from enemies and coins.
- **Upgrade / Shop menu**: Spend gold on stat upgrades, weapons, armor, and arrow bundles.
- **Interactive world**: Doors and chests.
- **Camera system**: Smooth camera offset with layered drawing (floor/main/overhead/doors).
- **Retro pixel art feel**: Classic pixel style with simple UI and effects.

## Installation

To run **Knight vs Sceletors**, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Seweryn-L/Roguelike_game_Python
   cd Roguelike_game_Python
2. **Install Python**: Ensure you have Python 3.8 or later installed. You can download it from python.org.

3. **Install Pygame**: Install the Pygame library using pip.
    ```bash
    pip install pygame
    pip install pytmx

4. **Run the Game**: Navigate to the project directory and run the Main.py file.
    ```bash
    python Main.py

## How to Play

**Movement**: Use the W, A, S, D keys to move your Knight.

**Change weapon**: You can change your weapon from your inventory by using Q and E.  

**Attack**: While you have melee weapon, click left button of your mouse, while you have bow aim with mouse and click left button to shoot. .

**Shop**: After each room or even more often, visit the shop to spend your money on upgrades or new weapons.

**Survive**: Kill as many enemies as you can, go through all rooms and open chests.
